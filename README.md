# 🗣️ PECSコミュニケーションボード

絵カード交換式コミュニケーションシステム（PECS）のWebアプリケーションです。
言葉でのコミュニケーションが難しい方のための、視覚的で直感的なコミュニケーションツールです。

## ✨ 主な特徴

### 🎯 コミュニケーション機能
- **カード選択で文章作成** - 絵カードを並べて文章を作成
- **音声読み上げ** - 作成した文章を自然な日本語で読み上げ
- **即座の再生** - ボタンを押して1秒以内に再生開始
- **視覚的ハイライト** - 現在読み上げ中のカードを強調表示

### 🎨 カスタマイズ機能
- **3種類の画像タイプ**
  - 📱 絵文字（手軽でカラフル）
  - 📷 写真アップロード（実物の写真が使える）
  - ✏️ 手描き（オリジナルイラスト作成）
- **柔軟なカテゴリ管理** - 自由にカテゴリを追加・編集
- **ピン留め機能** - よく使うカードを上部に固定
- **使用頻度カウント** - 自動で使用回数を記録

### 👶 2つのモード
- **子どもモード** - ひらがな表示で見やすい
- **大人モード** - 漢字表示でより詳細に

### 📊 データ管理
- **エクスポート/インポート** - JSONファイルで簡単バックアップ
- **複数のエクスポートオプション**
  - すべてのカード
  - ピン留めカードのみ
  - 選択したカテゴリのみ
- **オフライン動作** - データはブラウザに保存

### 🔧 便利な機能
- **並び替え** - 名前順、カテゴリ順、使用頻度順
- **ピンフィルター** - ピン留めカードだけを表示
- **カード検索** - すぐに目的のカードを発見
- **レスポンシブデザイン** - スマホ、タブレット、PCに対応

## 🚀 使い方

### 初めての起動

1. **ブラウザで開く**
   - iPhoneの場合：Safari推奨
   - Androidの場合：Chrome推奨
   - PCの場合：Chrome、Edge、Firefox等

2. **初期カードを確認**
   - アプリには最初からサンプルカードが用意されています
   - そのまま使うことも、編集・削除も可能です

### カードの作成

1. **「作る」ボタンをタップ**
2. **画像を選択**
   - 絵文字：プリセットから選択
   - 写真：カメラで撮影 or ギャラリーから選択
   - 描く：指やペンで自由に描画
3. **文字を入力**
   - **漢字**：大人モード（漢字表示）で表示される文字
   - **ひらがな**：子どもモード（ひらがな表示）で表示される文字（省略可）
   - **読み上げ**：音声で読まれるテキスト（ひらがな推奨、省略可）
   - ※ひらがなを省略すると漢字が使われます
   - ※読み上げを省略するとひらがなが使われます
4. **カテゴリを選択**
5. **「保存」をタップ**

### 文章の作成と読み上げ

1. **カードをタップ** - 下部のセンテンスストリップに追加
2. **順番に並べる** - 文章の語順に配置
3. **「おはなし」ボタン** - 音声で読み上げ
4. **修正する場合**
   - カードをタップ：その位置から削除
   - 「1つ消す」：最後のカードを削除
   - 「全消し」：全部クリア

### カテゴリ管理

#### カテゴリの追加
1. **設定ボタン（⚙️）をタップ**
2. **「カテゴリを編集」ボタンをタップ**
3. **新しいカテゴリを追加セクションで入力**
   - 絵文字：カテゴリを表すアイコン（例：🎮）
   - カテゴリ名：漢字表記（例：ゲーム）
   - ひらがな名：ひらがな表記（例：げーむ）
4. **「追加」ボタンをタップ**

#### カテゴリの編集
1. **カテゴリ管理画面を開く**
2. **編集したいカテゴリの✏️ボタンをタップ**
3. **絵文字、名前、ひらがな名を順に入力**
4. **確定**

#### カテゴリの削除
1. **カテゴリ管理画面を開く**
2. **削除したいカテゴリの🗑️ボタンをタップ**
3. **確認ダイアログで「OK」**
   - そのカテゴリのカードは自動的に「その他」に移動されます
   - デフォルトカテゴリ（必要、生活、食事、遊び、気持ち、色、その他）は削除できません

#### デフォルトカテゴリ
アプリには以下のカテゴリが最初から用意されています：
- ⭐ **必要** - トイレ、飲む、休むなど基本的な要求
- 🏠 **生活** - おはよう、ありがとうなど日常の言葉
- 🍎 **食事** - 食べ物、飲み物
- ⚽ **遊び** - 遊び、場所、活動
- 😊 **気持ち** - 感情表現
- 🎨 **色** - 色の名前
- 📦 **その他** - 分類されないもの

## 📱 インストール方法

### GitHub Pagesで公開する場合

1. **GitHubアカウント作成**
   - [GitHub](https://github.com)で無料アカウント作成

2. **リポジトリ作成**
   - 「New repository」をクリック
   - 名前を入力（例：`pecs-board`）
   - 「Public」を選択
   - 「Create repository」

3. **ファイルアップロード**
   - 「uploading an existing file」をクリック
   - `index.html` をドラッグ&ドロップ
   - 「Commit changes」

4. **GitHub Pages有効化**
   - Settings → Pages
   - Source: 「Deploy from a branch」
   - Branch: 「main」→「/ (root)」
   - 「Save」

5. **アクセス**
   - `https://ユーザー名.github.io/pecs-board/`
   - 数分後に利用可能

### iPhoneでアプリのように使う

1. **Safariで開く**
2. **共有ボタン（□↑）をタップ**
3. **「ホーム画面に追加」**
4. **名前を入力して「追加」**
5. **ホーム画面にアイコンが追加されます！**

## 💾 データのバックアップ

### エクスポート（保存）
1. 設定ボタン（⚙️）をタップ
2. 「エクスポート」を選択
3. エクスポート範囲を選択
   - すべてのカード
   - ピン留めのみ
   - 選択カテゴリ
4. ダウンロードされたJSONファイルを保管

### インポート（復元）
1. 設定ボタン（⚙️）をタップ
2. 「インポート」を選択
3. 保存したJSONファイルを選択
4. 「マージ」または「置き換え」を選択

## 🎨 カスタマイズのヒント

### 効果的なカード作成
- **写真は実物を使う** - 認識しやすい
- **3つの文字を使い分け**
  - 漢字：大人が見やすい正式な表記
  - ひらがな：子どもが読みやすい表記
  - 読み上げ：自然に聞こえる音声用の文字
- **読み上げはひらがな** - 自然な発音に
- **カテゴリで整理** - 探しやすく
- **よく使うものはピン留め** - すぐアクセス

### カテゴリの例
デフォルト以外にも、自由にカテゴリを追加できます：
- 🎮 ゲーム - ビデオゲーム、ボードゲーム
- 🎵 おんがく - 楽器、歌、音楽関連
- 🍽️ たべもの - より詳細な食べ物分類
- 👕 ふく - 衣類、アクセサリー
- 🏥 びょういん - 医療関連の言葉
- 🏫 がっこう - 学校関連の言葉
- 👨‍👩‍👧‍👦 かぞく - 家族、人の名前
- 🚗 のりもの - 車、電車、飛行機
- 🐶 どうぶつ - ペット、動物園の動物
- ⏰ じかん - 時間、スケジュール関連
- 📚 べんきょう - 学習、宿題関連

**ヒント**: カテゴリは多すぎると探しにくくなります。5〜15個程度が使いやすいです。

### 3つのフィールドの使い分け例
**例1: トイレのカード**
- 漢字：「トイレに行きたい」
- ひらがな：「といれにいきたい」
- 読み上げ：「といれにいきたいです」

**例2: 簡単な単語**
- 漢字：「おやつ」
- ひらがな：（空欄でOK、自動的に「おやつ」が使われる）
- 読み上げ：（空欄でOK、自動的に「おやつ」が使われる）

**例3: 丁寧な表現**
- 漢字：「ジュース」
- ひらがな：「じゅーす」
- 読み上げ：「じゅーすがほしいです」

### 文章例
- 「おかあさん」+「おやつ」+「がほしい」
- 「トイレ」+「いきたい」
- 「おんがく」+「きく」+「がほしい」

## 🔧 トラブルシューティング

### 音声が再生されない
- **デバイスの音量を確認**
- **サイレントモード解除**（iPhone）
- **ブラウザの権限確認**
- **別のブラウザで試す**

### カードが表示されない
- **カテゴリフィルター確認** - 「すべて」を選択
- **ページを再読み込み**
- **ブラウザキャッシュをクリア**

### データが消えた
- **同じブラウザで開いているか確認**
- **プライベートモードではないか確認**
- **バックアップから復元**

### 写真がアップロードできない
- **カメラ/ストレージのアクセス許可確認**
- **ファイルサイズが大きすぎないか確認**（推奨：5MB以下）
- **対応形式確認**（JPG、PNG）

## 📖 用語集

- **PECS（ペクス）**: Picture Exchange Communication System（絵カード交換式コミュニケーションシステム）
- **センテンスストリップ**: 文章を作るために選んだカードを並べる場所
- **ピン留め**: よく使うカードを一覧の上部に固定する機能
- **エクスポート**: データをファイルとして保存すること
- **インポート**: 保存したファイルからデータを読み込むこと
